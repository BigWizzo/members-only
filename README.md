## Getting Started
- To get a local copy up and running follow these simple steps.

### Install
- yarn
- Ruby
- Ruby on Rails
- Text Editor

### Instructions
- Open the console
- Download or `git clone https://github.com/BigWizzo/members-only.git`
- `cd members-only`
- run `bundle install`
- run `yarn install`
- run `rails server`
